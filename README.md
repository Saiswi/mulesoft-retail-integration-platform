# MuleSoft Retail Integration Platform

## Overview
This is an end-to-end real-time MuleSoft integration project built using API-led connectivity.
The project simulates a retail enterprise integration involving order management, inventory,
payments, Salesforce synchronization, batch processing, security, testing, and CloudHub deployment.

## Architecture
- Experience API
- Process API
- System APIs (Salesforce, Database, SFTP, Payment)

## Technologies Used
- Mule 4
- Anypoint Platform
- RAML 1.0
- DataWeave 2.0
- Salesforce Connector
- Database Connector
- SFTP
- MUnit
- CloudHub

## Key Features
- API-led architecture
- Production-grade error handling
- Secure APIs (OAuth / Client ID enforcement)
- Batch processing with watermarking
- CI/CD-ready deployment
